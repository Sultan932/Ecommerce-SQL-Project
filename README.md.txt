# Ecommerce SQL Project

## Project Overview
This is a beginner-friendly SQL project for practicing database design, data insertion, and queries.  
The project simulates a small e-commerce store with customers, products, orders, and order items.

## Objectives
- Create and manage a relational database
- Practice SELECT, WHERE, ORDER BY, LIMIT
- Perform aggregation (COUNT, SUM, AVG, GROUP BY)
- Use JOINs to connect tables
- Apply business logic using CASE statements
- Build reusable SQL Views
- Write subqueries for analysis

## Database Schema
- `customers`: Customer information
- `products`: Product information
- `orders`: Customer orders
- `order_items`: Items within orders

## Sample Queries
- Total revenue per product
- Top 3 customers by spending
- Monthly revenue trends
- Customer segmentation (VIP, Regular, Low Value)
- Customers with no orders

## Tools
- MySQL
- MySQL Workbench

## How to Use
1. Import `database.sql` into MySQL.
2. Run `queries.sql` to see sample analyses.
3. Explore by modifying queries or adding more data.
